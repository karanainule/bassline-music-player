# 🎵 Bassline Music Player  

A simple **Spotify-like music player** built using HTML, CSS, and JavaScript.  

## 🚀 Features
- Play / Pause songs 🎶  
- Next / Previous track controls ⏭️⏮️  
- Playlist support 📂  
- Responsive design 📱  
- Dark theme UI 🎨  

---

bassline-music-player/
│-- index.html # Main page
│-- style.css # Styling
│-- script.js # Music player logic
